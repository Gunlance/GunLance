实现最小栈

**想法**

多push一个最小数咯

### 法一

```py
class MinStack:

    def __init__(self):
        """
        initialize your data structure here.
        """
        self.min = float("inf")
        self.stack = []

    def push(self, x: int) -> None:
        if x <= self.min:
            self.stack.append(self.min)
            self.min = x
        self.stack.append(x)

    def pop(self) -> None:
        if self.stack.pop() <= self.min:
            self.min = self.stack.pop()

    def top(self) -> int:
        return self.stack[-1]
    
    def getMin(self) -> int:
        return self.min
```
